=== OAI Chatbox Pro ===
Contributors: OpenAI
Requires at least: 6.0
Tested up to: 6.8
Requires PHP: 7.4
Stable tag: 1.0.0
License: GPLv2 or later

A modern floating chat box for WordPress with visitor chat history, admin inbox, tags, assignment, notes, rating, CSV export, offline form, typing indicator, and optimized polling realtime.

== Included in this build ==
- Floating responsive chat widget
- Visitor pre-chat form (optional)
- Chat history persisted by conversation token in browser
- Optimized polling realtime updates
- Typing indicator and read status support
- Greeting message and first-message auto reply
- Online/offline business-hours handling
- Offline leave-a-message flow
- File upload with allowlist + size limit
- Admin inbox UI with filters, notes, tags, assignment
- CSV export
- Basic email alert for new chats
- Security basics: nonce, capability checks, sanitize/validate, file extension allowlist, rate limiting

== Notes ==
This package ships as a strong practical MVP. It uses optimized polling instead of WebSocket/Pusher.
Native integrations such as Facebook Messenger, Telegram, Zalo, CRM sync, AI reply suggestions, and voice messages are not included in this v1 package.
