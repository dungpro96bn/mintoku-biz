== Description ==

Adds reCAPTCHA field to MW WP Form.


== Installation ==

1. Upload the `mw-wp-form-recaptcha` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the Plugins menu in WordPress.
3. Add the Site Key.
